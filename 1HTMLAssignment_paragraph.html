USING THE INTERNET CONNECTION WIZARD

Connecting to the Internet is quick and easy using the Internet Connection Wizard. The Internet Connection Wizard helps you sign up an account

To start the wizard for the first time, double-click The Internet icon on your desktop. If you need to start an wizard again at a later time. click the Start menu, point to Programs, point to Accessories, point to Internet Tools, and then click Get On The Internet


The wizard connects you to the Microsoft Internet Referral Service. This service provides you with a list of the service providers available in your area. To determine which service providers are available, the wizard sends the following information to the Internet Referral Service.


      Field                        Description

Country Id           The Country from which you are dialling

Area/City Code   Your area or city code
